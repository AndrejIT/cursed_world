cursed_world
=========

Mod "cursed_world" for Minetest 4.14+ game.

Allows players to build portals by placing 10 obsidian blocks and then adding 2 "cursed_stone".
Portal output then becomes connected to distant location.

This distant location - "cursed world" is not included by this mod,
and must be created and added to minetest map by some other means.
